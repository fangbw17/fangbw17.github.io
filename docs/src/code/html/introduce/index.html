HTML 简介

HTML 是用来描述网页的一种语言。

HTML 指的是超文本标记语言: HyperText Markup Language

HTML 不是一种编程语言，而是一种标记语言

标记语言是一套标记标签 (markup tag)

HTML 使用标记标签来描述网页

HTML 文档包含了 HTML 标签及文本内容

HTML 文档也叫做 web 页面

你可以通过网页浏览器（如 Internet Explorer、Firefox、Chrome、Safari 等）来请求 HTML 文档。

请看下面的例子:

例子

下面的 HTML 文档包含了一个标题 (heading) 和两个段落 (paragraph):

      <!DOCTYPE html>
      <html>
      <head>
      <title>我的第一个 HTML 页面</title>
      </head>
      <body>
      <h1>我的第一个标题</h1>
      <p>我的第一个段落。</p>
      <p>我的第二个段落。</p>
      </body>
      </html>

上面的例子中，<!DOCTYPE html> 声明了文档类型，<html> 元素是 HTML 文档的根元素，<head> 元素包含了文档的元（meta）数据，<title> 元素设置了文档的标题，<body> 元素包含了可见的页面内容，<h1>
